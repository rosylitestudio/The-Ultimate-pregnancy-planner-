## Common Questions

**Q: Where is my data?**
A: All on your device. Never sent anywhere. 100% private.

**Q: Can I access on my phone?**
A: Yes! Open the HTML file in mobile browser. All features work perfectly.

**Q: Will my data disappear?**
A: No. Stored locally in your browser. Never deleted unless you clear browser storage (very rare).

**Q: What if my browser crashes?**
A: Data is saved. Open file again. Everything is there.

**Q: Can I print it?**
A: Yes! Export any tab as PDF, then print for your planner or hospital bag.

**Q: How do I backup?**
A: Click ⚙️ (settings) → Export Data. Save JSON file to Google Drive or email.

**Q: Can my partner use it?**
A: Give them the HTML file. They'll have their own separate copy (no sync).

**Q: Is there a mobile app?**
A: Not needed! Browser works perfectly as an app.

**Q: Will this work after my due date?**
A: Yes, forever. No subscriptions. No expiration date. Works for postpartum too.

**Q: What if I need help?**
A: Check the built-in "?" help button for tutorials. Or contact seller support.
